# Example User Prompts

## Basic Caption Request
```
Add captions to my video with word highlighting
```

## Detailed Caption Request
```
Add captions to my video using Montserrat Bold font, size 64, with yellow text that turns cyan when spoken. Position captions at the bottom.
```

## Multilingual Request
```
Transcribe and caption this Spanish video with white text on black outline, karaoke-style word highlighting
```

## Custom Styling
```
Create TikTok-style captions - bold Impact font, large size, white text with thick black outline, bottom position, and color change highlight from white to yellow
```

## Format Examples

### Minimal
"Add captions to video.mp4"

### With Style
"Add captions with Arial Bold, medium size, color_change highlight (white to cyan)"

### Full Specification
"Transcribe video.mp4, add captions with: font=Roboto Bold, size=large, position=bottom, primary_color=white, highlight_color=yellow, highlight_type=karaoke_fill, outline_width=4, outline_color=black"
