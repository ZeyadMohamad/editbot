## CAPTION PIPELINE INSTRUCTIONS

When user requests captions, subtitles, or transcription overlay:

### REQUIRED STEPS (in order)
1. **extract_audio** - Extract audio track from video
2. **transcribe** - Convert speech to text with timestamps
3. **align_words** - Get precise word-level timing
4. **generate_captions** - Create styled ASS subtitle file
5. **render_subtitles** - Burn subtitles onto video

### STYLE PARAMETERS TO EXTRACT FROM PROMPT
Look for these in the user's request:
- **Font**: Font family name (e.g., "Montserrat", "Arial Bold")
- **Size**: tiny/small/medium/large/extra_large/huge OR pixel value
- **Primary Color**: Text color when not highlighted
- **Highlight Color**: Text color when word is being spoken
- **Highlight Type**: `word_by_word` or `progressive`
- **Current Word Bold**: whether active word should become bold
- **Current Word Box**: background box behind active word
- **Position**: bottom/top/middle/center + left/right
- **Outline**: Width and color of text border
- **Shadow**: Shadow depth

### HIGHLIGHT INPUT OBJECT (generate_captions)
Use `highlight_options` in job inputs:
- `enabled`: true/false
- `highlight_type`: `word_by_word` or `progressive`
- `highlight_color`: ASS BGR hex (e.g., `00FFFF`)
- `progressive_color`: ASS BGR hex
- `current_word_bold`: true/false
- `current_word_box`: true/false
- `box_color`: ASS BGR hex
- `box_alpha`: two-digit ASS alpha

### LAYERING RULE
- Layer 0: full caption sentence (base text).
- Layer 1: timed highlight overlay.
- Both layers should start from the right side of the video.

### DEFAULT VALUES (when not specified)
- Font: Montserrat ExtraBold
- Size: medium (48px)
- Primary Color: white
- Highlight Color: yellow
- Highlight Style: color_change
- Position: bottom_center
- Outline: 3px black
- Shadow: 2

### STYLE EXTRACTION EXAMPLES

User: "Add yellow captions with TikTok style"
Extract:
- highlight_color: yellow
- Use tiktok_bold preset (if available)

User: "Transcribe with Arial font, large size, karaoke fill effect"
Extract:
- font: Arial Bold
- size: large
- highlight_style: karaoke_fill

User: "Add subtitles at the top in red"
Extract:
- position: top_center
- primary_color: red

### LANGUAGE HANDLING
- If user specifies language, use it
- Otherwise, set language to "auto" for auto-detection
- Whisper handles multilingual content automatically
