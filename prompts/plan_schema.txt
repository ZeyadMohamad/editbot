## PLAN JSON SCHEMA

The LLM must output plans conforming to this structure:

### Top-Level Response
```json
{
  "success": boolean,          // Whether plan was created successfully
  "plan": Plan | null,         // Execution plan (null if success=false)
  "missing_inputs": string[],  // Required inputs not provided
  "validation_errors": string[], // Config validation failures
  "message": string            // Human-readable status message
}
```

### Plan Object
```json
{
  "plan_id": string,           // Unique identifier (e.g., "plan_20240128_001")
  "intent": string,            // Summary of what user wants
  "jobs": Job[],               // Ordered list of jobs
  "estimated_duration": number // Optional: estimated seconds
}
```

### Job Object
```json
{
  "job_id": string,            // Unique within plan (e.g., "job_1")
  "job_type": string,          // Tool name from registry
  "inputs": {                  // Tool-specific inputs
    "param_name": value
  },
  "outputs": {                 // Expected output artifacts
    "artifact_name": "path_or_placeholder"
  },
  "depends_on": string[]       // Job IDs this depends on
}
```

### Valid job_type Values
- extract_audio
- get_video_info
- transcribe
- align_words
- generate_captions
- render_subtitles
- silence_cutter
- stock_footage
- apply_transitions

### Input/Output Placeholders
Use these placeholders for dynamic paths:
- {input_video}: User's input video path
- {workspace}: Job workspace directory
- {job_id}: Current job's ID
- {output_dir}: Final output directory

### Manual Cut Segments (silence_cutter)
If the user provides exact timestamps, include:
```json
"manual_cut_segments": [{"start": 18.005, "end": 18.670}]
```

### Stock Footage (stock_footage)
Use `stock_items` to describe overlays or insertions:
```json
"stock_items": [
  {
    "path": "C:/assets/broll.mp4",
    "mode": "insert",
    "start_time": 42.0,
    "end_time": 46.0,
    "source_start": 3.0,
    "source_end": 7.0
  },
  {
    "path": "C:/assets/chart.png",
    "mode": "overlay",
    "start_time": 12.0,
    "end_time": 18.0,
    "position": "top_right",
    "size": "40%",
    "margin": "5%"
  }
]
```

### Transitions (apply_transitions)
Use `clips` to provide ordered clip paths and `transitions` to describe each cut:
```json
"clips": [
  {"path": "C:/assets/clip1.mp4"},
  {"path": "C:/assets/clip2.mp4", "start_time": 2.0, "end_time": 9.5},
  {"path": "C:/assets/clip3.mp4"}
],
"transitions": [
  {"name": "Cross Dissolve", "duration": 1.0},
  {"name": "Dip to Black", "duration": 0.8}
],
"transition_duration": 1.0
```

For transitions inside a single clip, use `segments` with a shared `source_path`:
```json
"source_path": "C:/assets/long_clip.mp4",
"segments": [
  {"start_time": 0.0, "end_time": 6.0},
  {"start_time": 8.0, "end_time": 14.5},
  {"start_time": 18.0, "end_time": 26.0}
],
"transitions": [
  {"name": "Cross Dissolve", "duration": 0.8},
  {"name": "Wipe Right", "duration": 0.6}
]
```

### Example Complete Plan
```json
{
  "success": true,
  "plan": {
    "plan_id": "plan_caption_001",
    "intent": "Add word-highlighted captions to video",
    "jobs": [
      {
        "job_id": "job_1",
        "job_type": "extract_audio",
        "inputs": {
          "video_path": "{input_video}",
          "output_path": "{workspace}/audio.wav",
          "sample_rate": 16000
        },
        "outputs": {
          "audio_file": "{workspace}/audio.wav"
        },
        "depends_on": []
      },
      {
        "job_id": "job_2",
        "job_type": "transcribe",
        "inputs": {
          "audio_path": "{workspace}/audio.wav",
          "language": "auto"
        },
        "outputs": {
          "transcript": "{workspace}/transcript.json"
        },
        "depends_on": ["job_1"]
      },
      {
        "job_id": "job_3",
        "job_type": "align_words",
        "inputs": {
          "transcription_result": "{job_2.outputs.transcript}"
        },
        "outputs": {
          "words": "{workspace}/words.json"
        },
        "depends_on": ["job_2"]
      },
      {
        "job_id": "job_4",
        "job_type": "generate_captions",
        "inputs": {
          "words": "{job_3.outputs.words}",
          "output_path": "{workspace}/captions.ass",
          "style": {
            "font": "Montserrat ExtraBold",
            "font_size": 48,
            "primary_color": "FFFFFF",
            "position": "bottom_center"
          },
          "highlight_options": {
            "enabled": true,
            "highlight_type": "progressive",
            "highlight_color": "00FFFF",
            "progressive_color": "00FFFF",
            "current_word_bold": true,
            "current_word_box": true,
            "box_color": "000000",
            "box_alpha": "66"
          }
        },
        "outputs": {
          "ass_file": "{workspace}/captions.ass"
        },
        "depends_on": ["job_3"]
      },
      {
        "job_id": "job_5",
        "job_type": "render_subtitles",
        "inputs": {
          "video_path": "{input_video}",
          "subtitle_path": "{workspace}/captions.ass",
          "output_path": "{output_dir}/output.mp4"
        },
        "outputs": {
          "video_file": "{output_dir}/output.mp4"
        },
        "depends_on": ["job_4"]
      }
    ]
  },
  "missing_inputs": [],
  "validation_errors": [],
  "message": "Plan created successfully with 5 jobs"
}
```
